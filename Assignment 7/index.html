Lab 01

Attributes vs Properties

HTML Attributes represent the source document's initial state. DOM Properties represent the live state of elements in browser memory.

Property: input.value
"Initial HTML Value"
Live DOM state in memory (updates instantly as you type).
Attribute: input.getAttribute("value")
"Initial HTML Value"
Original HTML markup configuration (stays constant unless setAttribute called).
